# 🚀 API com Flask + Ambiente Virtual Python

Projeto desenvolvido para estudos de APIs REST utilizando Flask e ambientes virtuais em Python.


## 📖 Sobre o Projeto

Esta aplicação foi criada com o objetivo de aprender conceitos importantes do desenvolvimento backend com Python, incluindo:

- Criação de APIs REST

- Utilização do Flask

- Ambientes virtuais (venv)

- Rotas HTTP

- Requisições e respostas JSON

- Organização de projetos Python


 
## 🛠️ Tecnologias Utilizadas
- Python 3.10+
- FastAPI
- Flask
- Virtual Environment (venv)
- Requests
- Git & GitHub


## 📂 Estrutura do Projeto
Venv/

## ⚙️ Como Executar o Projeto

1️⃣ Clone o repositório
git clone https://github.com/rayssavilela/VENV-Python.git

2️⃣ Acesse a pasta do projeto
cd VENV-Python

3️⃣ Crie o ambiente virtual
python -m venv venv

4️⃣ Ative o ambiente virtual

- Windows
venv\Scripts\activate

- Linux/Mac
source venv/bin/activate

5️⃣ Instale as dependências
pip install -r requirements.txt

6️⃣ Execute a aplicação
python app.py
